# ColorRemesh

Приложение для перевода цветов по коду/колесу цветов/ползуноков параметров
CMYK <> Lab <> HSV

# Запуск приложения через .py файл

Windows

```
    python -m venv .venv
    .venv/Scripts/acitvate
    pip install -r requirements.txt
    python main.py
```

MacOS & Linux

```
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    python3 main.py
```
